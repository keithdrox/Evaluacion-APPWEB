# Evaluacion APPWEB

Aplicación web de evaluación desarrollada con HTML, CSS, JavaScript y Webpack.

## Estructura del proyecto

- `index.html` - página principal.
- `css/` - estilos del proyecto.
- `js/` - scripts de la aplicación.
- `img/` - recursos de imagen.
- `webpack.common.js`, `webpack.config.dev.js`, `webpack.config.prod.js` - configuración de Webpack.

## Uso

1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Ejecutar en modo desarrollo:
   ```bash
   npm run dev
   ```
3. Generar producción:
   ```bash
   npm run build
   ```

## Nota

Este repositorio contiene una aplicación web estática con configuración de bundling mediante Webpack.
